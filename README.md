Projeto para apoio na checagem de preços em farmácia, pensado para uso interno no escritório. A ideia é organizar a proposta, documentar o funcionamento e manter o projeto preparado para evoluções futuras.

## Objetivos

- Facilitar consultas de preços.
- Reduzir trabalho repetitivo em tarefas internas.
- Registrar o contexto e a evolução do projeto.
- Manter documentação clara para manutenção.
