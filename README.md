# BuscaFarma

Projeto para apoio na checagem de precos em farmacia, pensado para uso interno no escritorio. A ideia e organizar a proposta, documentar o funcionamento e manter o projeto preparado para evolucoes futuras.

## Objetivos

- Facilitar consultas de precos.
- Reduzir trabalho repetitivo em tarefas internas.
- Registrar o contexto e a evolucao do projeto.
- Manter documentacao clara para manutencao.

## Status

Projeto em organizacao e evolucao.
